Capstone Project

Response VCS System

A mobile application will be built to operate on Android Operating System. This application will enable users to send GPS location and textual information as distress alert to a Postgresql database through Representative State Transfer (RESTful) request and response Application Programming Interface (API). The spatially enabled database will then performs a ten miles’ radius proximity analysis (Buffer) to select users in this radius. The alert will then be dispatched to these users as a push notification. Users will response to this alert with either the proposed built in navigational tab that will be built in the mobile application which will extend openstreetmap as a basemap or use the Maps navigation application provided by google on android operating system. A web mapping application will be created to visualize and download responses data in a browser.

  • Date: December 18, 2020
  • By: myLORD
